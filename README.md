# SWC-HTML-CSS-JS-project

SWC-HTML-CSS-JS-project is a webpage that provides users with the latest movie details and the ability to search through them. It utilizes the MoviesDB API to fetch movie data and display it on the webpage.

## Installation

1. Clone the repository to your local machine.
   ```bash
   git clone https://github.com/Sai-004/SWC-HTML-CSS-JS-project.git
   ```

2. Open the project folder.
   ```bash
   cd SWC-HTML-CSS-JS-project
   ```

3. Open the `index.html` file in a web browser.

## Usage

The webpage allows users to perform the following actions:

- View the latest movie details: The webpage fetches data from the MoviesDB API and displays the latest movie details, including the title, release date, overview, and poster image.

- Search for movies: Users can enter a search query in the search bar to find movies that match the query. The webpage then fetches the movie data based on the search query and displays the results.

- View movie reviews: Users can click on a movie card to view detailed information about the movie, including reviews and ratings from critics and users.

## API Used

The project utilizes the MoviesDB API to retrieve movie data. You can find more information about the API and how to use it on the [MoviesDB website](https://www.themoviedb.org/documentation/api).
